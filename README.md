# NODEJS

## 1. 노드의 정의

- Node.js는 크롬 V8 자바스크립트 엔진으로 빌드된 자바스크립트 런타임(실행기)입니다.
- 노드는 싱글 스레드 모델에 비동기 I/O

### 이벤트 기반

- 이벤트가 발생할 때 미리 지정해둔 작업을 수행하는 방식
  · 이벤트의 예: 클릭, 네트워크 요청, 타이머 등
  · 이벤트 리스너: 이벤트를 등록하는 함수
  · 콜백 함수: 이벤트가 발생했을 떄 실행될 함수
